# Medical Image Replacement Instructions

## 🎯 How to Add Your Medical Images

### 📁 Current Placeholder Files:
- `radiyug-cardiac-3d.jpg` - Placeholder for your **enhanced Radiyug quality** medical image
- `traditional-cardiac-axial.jpg` - Placeholder for your **traditional printer quality** medical image

### 🖼️ Replace with Your Images:

1. **For Radiyug Enhanced Image:**
   - Save your **colorful 3D cardiac reconstruction** image (first attachment) as:
   - `radiyug-cardiac-3d.jpg`
   - This should be the **high-quality, color-enhanced** medical image

2. **For Traditional Quality Image:**
   - Save your **grayscale axial cardiac** image (second attachment) as:
   - `traditional-cardiac-axial.jpg`
   - This should be the **standard quality, grayscale** medical image

### ⚙️ Image Specifications:
- **Format**: JPEG (.jpg)
- **Recommended Size**: 600x400 pixels (auto-resized)
- **Quality**: High resolution for medical accuracy
- **File Size**: Optimized for web (under 500KB recommended)

### 🎨 Auto-Sizing Features:
- Images automatically resize for different screen sizes
- Responsive design: Mobile, Tablet, Desktop
- Optimized loading with Next.js Image component
- Quality settings: Radiyug (90%), Traditional (75%)

### 🔄 After Replacement:
1. Save your images with the exact filenames above
2. Refresh the browser (the website will automatically load your images)
3. The comparison animation will show your actual medical data!

### 📊 Component Features:
- **Animated comparison** between your actual images
- **Quality filters** applied automatically
- **Responsive sizing** for all devices
- **Error handling** if images fail to load
- **Performance optimized** with Next.js Image optimization
